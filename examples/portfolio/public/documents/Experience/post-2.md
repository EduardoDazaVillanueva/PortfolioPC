# Prácticas de Grado Superior – Controlnet Web S.L.
**Periodo:** Marzo - Junio 2024

Durante mis prácticas de grado superior en Controlnet Web S.L., participé en proyectos de desarrollo web y análisis de datos, mejorando la eficiencia y experiencia de usuario en aplicaciones internas.

- **Análisis y visualización de datos mediante Qlik:** Extraje y procesé datos para generar informes visuales que facilitaron la toma de decisiones del equipo.  
- **Desarrollo de aplicaciones web con React:** Implementé funcionalidades interactivas y mejoré la experiencia de usuario en aplicaciones existentes, siguiendo buenas prácticas de programación y diseño.

Estas prácticas reforzaron mis competencias en desarrollo frontend, manejo de datos y trabajo colaborativo en un entorno profesional.
